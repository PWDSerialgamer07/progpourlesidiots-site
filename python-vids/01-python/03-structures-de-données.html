03 Structures de données

Étape 1: Listes
...

Une liste est une collection ordonnée et modifiable d'éléments. Les éléments d'une liste sont définis entre crochets [] et séparés par des virgules.

1
# Création d'une liste
2
nombres = [1, 2, 3, 4, 5]

Étape 2: Dictionnaires
...

Un dictionnaire est une collection non ordonnée, modifiable et indexée d’éléments. Les éléments d’un dictionnaire sont des paires clé-valeur définies entre accolades {}.

Python

1
# Création d'un dictionnaire
2
personne = {
3
    "nom": "Alice",
4
    "age": 25,
5
    "ville": "Paris"
Best practices
  • Utilisez des listes pour stocker une collection d’éléments qui doivent être ordonnés ou qui peuvent avoir des doublons.
  • Utilisez des dictionnaires pour stocker des données sous forme de paires clé-valeur lorsque vous avez besoin d’un accès rapide à un élément spécifique.
