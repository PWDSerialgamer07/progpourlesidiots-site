Étape 1: Variables et Types de Données
...

En Python, une variable est un conteneur pour stocker des données. Vous pouvez imaginer une variable comme une boîte dans la mémoire de l'ordinateur qui stocke une certaine valeur.

1
# Déclaration d'une variable
2
x = 10

Python a plusieurs types de données intégrés. Par exemple :

  • int : pour représenter des nombres entiers (ex: 10, 2, 78)
  • float : pour représenter des nombres réels (ex: 0.9, 15.12)
  • str : pour représenter des chaînes de caractères (ex: “Bonjour”, “Python”)
  • bool : pour représenter des valeurs booléennes (True ou False)

Étape 2: Opérations de base
...

Python supporte une variété d’opérations arithmétiques sur les nombres, comme l’addition (+), la soustraction (-), la multiplication (*) et la division (/).

1
# Exemples d'opérations arithmétiques
2
x = 10 + 5  # Addition
3
y = 10 - 5  # Soustraction
4
z = 10 * 2  # Multiplication
5
w = 10 / 2  # Division

Python fournit également des opérateurs de comparaison (==, !=, <, >, <=, >=) pour comparer des valeurs, et des opérateurs logiques (and, or, not) pour combiner des conditions.

Étape 3: Structures de contrôle
...

Les instructions conditionnelles en Python vous permettent de contrôler le flux d’exécution de votre programme. Les mots-clés if, elif et else sont utilisés pour cela.

1
# Exemple d'instruction conditionnelle
2
x = 10
3
if x > 0:
4
    print("x est positif")
5
elif x < 0:
6
    print("x est négatif")
7
else:
8
    print("x est zéro")
9

Python fournit également des boucles pour répéter une certaine partie du code. Les mots-clés for et while sont utilisés pour cela.

La boucle for en Python est utilisée pour itérer sur une séquence (qui peut être une liste, un tuple, un dictionnaire, un ensemble ou une chaîne).

1
# Exemple de boucle for
2
for i in range(5):
3
    print(i)

Cela affichera les nombres de 0 à 4, car range(5) génère une séquence de nombres de 0 à 4.

La boucle while en Python est utilisée pour répéter un bloc de code aussi longtemps qu’une condition est vraie.

1
# Exemple de boucle while
2
i = 0
3
while i < 5:
4
    print(i)
5
    i += 1

Cela affichera également les nombres de 0 à 4. La boucle continue de s’exécuter tant que i < 5 est vrai, et i est incrémenté de 1 à chaque itération.

Étape 4: Fonctions
...

Une fonction est un bloc de code réutilisable qui effectue une tâche spécifique. Vous pouvez définir vos propres fonctions en utilisant le mot-clé def.

1
# Définition d'une fonction
2
def saluer():
3
    print("Bonjour tout le monde!")
4

5
# Appel de la fonction
6
saluer()
7

Best Practices
  • Utilisez des noms de variables descriptifs pour rendre votre code plus lisible.
  • Essayez de garder vos fonctions courtes et concentrées sur une seule tâche. Cela rend votre code plus facile à lire et à déboguer.
  • Testez votre code souvent pendant le développement pour attraper les erreurs tôt.

Exercice à la maison

Votre tâche est de créer un programme qui demande à l'utilisateur d'entrer un nombre, puis utilise une boucle for pour afficher tous les nombres de 0 jusqu'à ce nombre. Alternativement, le programme peut utiliser une boucle while pour faire la même chose (je recommande de faire deux programme, un avec for et l'autre avec while).

De plus, le programme doit définir une fonction qui prend un nombre en entrée et retourne True si le nombre est pair et False sinon. Utilisez cette fonction dans vos boucles pour afficher si chaque nombre est pair ou impair.
N'oubliez pas, il n'y a pas qu'une seule réponse juste.
